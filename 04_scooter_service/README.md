# Исследование данных сервиса аренды самокатов GoFast

## Данные

Данные о некоторых пользователях из нескольких городов, а также об их поездках. 

## Задача

Необходимо проанализировать данные и проверить некоторые гипотезы, которые могут помочь бизнесу вырасти.

## Используемые библиотеки
*pandas, matplotlib.pyplot, numpy, scipy.stats, math*
