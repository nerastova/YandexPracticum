# Исследование данных сервиса “Яндекс.Музыка” — сравнение пользователей двух городов

## Данные

Реальные данные сервиса Яндекс.Музыка. 

## Задача

С помощью библиотеки Pandas и её возможностей проверить данные и сравнить поведение и предпочтения пользователей двух столиц — Москвы и Санкт-Петербурга.

## Используемые библиотеки
*pandas*
